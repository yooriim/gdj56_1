서버에서 데이터를 저장하는 객체

  • HttpServletRequest : 1회용 데이터를 저장하는 저장소 *1회용 -> 요청하고 응답이 완료될 때까지만 사용 (젤 마니 씀)
  • HttpSession : 원하는 시점까지 데이터를 유지하는 저장소
    *개발자가 삭제하거나 설정값에 의해 자동으로 삭제될 때까지 이용 (그 다음으로 마니 씀)
  • ServletContext : 서버가 실행되고 종료될 때까지 유지하는 저장소
